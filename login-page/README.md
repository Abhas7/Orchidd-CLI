# Login Page

This project provides a simple, clean, and functional login page built with HTML, CSS, and vanilla JavaScript. It includes client-side validation for username and password fields.

## Features

*   **Modern Design**: Clean and responsive layout with modern CSS.
*   **Client-Side Validation**: Basic validation for empty username and password fields.
*   **Error Handling**: Displays informative error messages for invalid inputs.
*   **User-Friendly**: Simple and intuitive form for a smooth user experience.

## Technologies Used

*   HTML5
*   CSS3
*   JavaScript (ES6+)

## Setup and Running

1.  **Clone the repository (or save the files locally)**:

    ```bash
    git clone https://github.com/your-username/login-page.git
    cd login-page
    ```

    *Alternatively, simply create a folder named `login-page` and place all the provided files inside it.*

2.  **Open `index.html`**: The application does not require a server or any build steps. Simply open the `index.html` file in your preferred web browser.

    You can do this by:

    *   Navigating to the `login-page` directory in your file explorer and double-clicking `index.html`.
    *   Using your browser's 'Open File...' option and selecting `index.html`.

## Usage

*   Enter a username and password into the respective fields.
*   Click the "Login" button.
*   If fields are empty, an error message will appear.
*   For demonstration purposes, successful login will log the credentials to the browser's console.
