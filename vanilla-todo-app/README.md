# Vanilla Todo App

A simple, clean, and responsive Todo application built with pure HTML, CSS, and JavaScript. This application allows users to add, mark as complete, and delete todo items, with all data persistently stored in the browser's local storage.

## Features

*   **Add Todos**: Quickly add new tasks to your list.
*   **Mark as Complete**: Toggle the completion status of a todo item with a click.
*   **Delete Todos**: Remove tasks you no longer need.
*   **Persistent Storage**: All your todos are saved in your browser's local storage, so they remain even after closing and reopening the browser.
*   **Clean UI**: A modern and user-friendly interface.
*   **Input Validation**: Prevents adding empty todo items.
*   **Confirmation on Delete**: Prompts user before deleting a task.

## Setup and Usage

This project does not require any build tools or complex setup. You can run it directly in your web browser.

1.  **Clone the repository (or download the files):**
    If you're getting these files from a repository, first clone it:
    ```bash
    git clone <repository-url>
    cd vanilla-todo-app
    ```
    If you've downloaded the files, just navigate to the `vanilla-todo-app` directory.

2.  **Open `index.html`:**
    Locate the `index.html` file in the project directory and open it with your preferred web browser (e.g., Chrome, Firefox, Edge). You can usually do this by double-clicking the file.

    Alternatively, from your terminal, you can use a command like:
    ```bash
    open index.html # On macOS
    start index.html # On Windows
    xdg-open index.html # On Linux
    ```

3.  **Start Using:**
    *   Type a task into the input field and click "Add Todo" (or press Enter) to add it to the list.
    *   Click on a todo item's text to mark it as complete or incomplete.
    *   Click the "Delete" button next to a todo item to remove it from the list.

## Technologies Used

*   **HTML5**: For the structure of the web page.
*   **CSS3**: For styling and making the application visually appealing and responsive.
*   **JavaScript (ES6+)**: For dynamic behavior, DOM manipulation, and local storage management.
